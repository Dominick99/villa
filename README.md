# VILLA — Virtual Immersion & Language Learning Agents

A language-learning prototype that simulates real-world immersion. The user picks a language (e.g. Russian) and a scenario (e.g. going on a date, ordering food, talking with locals as a green beret). An LLM agent generates an NPC template, then a separate LLM plays that character and converses with the user in the foreign language, forcing them to learn under pressure.
